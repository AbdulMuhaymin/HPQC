program __betafit.f__

Program to fit NTP read-in potential fx. values \( 0{RTP(i),VTP(i)} \) (with or without individual weights) to a chosen analytic form.

\( PSEL \) specifies the type of potential being fitted to: 

 PSEL = 1 for EMO
 PSEL = 2 for an MLR (or MLJ)
 PSEL = 3 for DELR
 PSEL = 4 for GPEF

\( NPT \) is the number of read-in potential points being fitted to.

\( UNC \) is the energy uncertainty associated with the potential points (plausibly ca. 0.1 cm-1 for RKR). To weight each point separately set UNC < 0.0 and read in a separate uncertainty for each point.

\( IROUND \) specifies the level of rounding inside NLLSSRR if: 

 > 0: requires that Sequential Rounding & Refitting be performed, with each parameter being rounded at the IROUND'th sig. digit of its local uncertainty.
 < or = 0: simply stops after full convergence (without rounding).

\( LPPOT \) specifies whether ( \( LPPOT \) > 0) or not ( \( LPPOT \leq 0 \)) the code should read instructions to generate an array of potential function values on some specified mesh and range. 

 If LPPOT > 0, read a separate mesh and range for each {N,p,q,Rref} case.

\( prFIT \) = 1 causes printout of results of preliminary linearized fit and (as appropriate) other non-final fits. Normally set \( prFIT = 0 \).

\( prFIT \) > 1 specifies the level of printing inside fit subroutine NLLSSRR. If: 

 >= 1  print converged parameters, PU & PS's.
 >= 2  also print parameter change each rounding step.
 >= 3  also indicate nature of convergence.
 >= 4  also print convergence tests on each cycle.
 >= 5  also parameters changes & uncertainties, each cycle.

\( prDIFF \) (integer) > 0 causes fit residuals to be printed after each complete fit: no print if \( prDIFF \leq 0 \).

\( Re \) & \( De \) are assumed potential well minimum position & well depth.

Note:

\( De \) is a dummy variable for a \( GPEF \) function. \( VMIN \) is the minimum of the potential function defined by the read-in points [typically = 0 for RKR potential, but non-zero for ab initio]. To fix \( Re \), \( De \) or \( VMIN \) unchanged at read-in values, set (integer) \( IFXRe \), \( IFXDe \), and/or \( IFXVMIN \) > 0 ; to fit them, set value =0 [Normally set IFXVMIN = 0 !!]

For an \( MLR (PSEL=2) \) or \( DELR (PSEL=3) \) potential, read the number of long-range terms NCMM used to define long-range potential tail:

\[ V(r)= De - \sum{\frac{CmVAL}{r^{MMLR}}} \]

If \( rhoAB \leq 0.0 \) have NO damping functions, all \( Dm(R) = 1.0 \).

If \( rhoAB \) > 0.0, it is the molecule-dependent radial scaling factor of Douketis et al. [JCP 76, 3057 (1982)]

\[ rhoAB = 2 \frac{rhoA*rhoB}{rhoA+rhoB} \]

where \( rhoA \) is the ionization potential ratio \( \left| \frac {I_p^A}{I_p^H}^{2/3} \right| \) for atom A vs. atomic H.

For \( rhoAB \) > 0.0, sVSR2 specifies damping such that:

\[ \frac{Dm(r)}{r^m} \rightarrow r^{\frac{sVSR2}{2}} \]

\( IDSTT \) > 0 use generalized Douketis et al. damping fx.

\( IDSTT \) > 1 use \( DS \) damping for \( s = -1/2 \) AND impose very short-range \( \frac{1}{r} \) behaviour with \( C1 = IDSTT = Z1*Z2 \).

\( IDSTT \leq 0 \), use generalized Tang-Toennies damping fx.

\( APSE \) is an integer: 

 > 0: use A.Pashov natural Spline in MLR Exponent.
 < or = 0: for normal constrained polynomial exponent.

For an \( MLR \) potential with \( APSE \) > 0, \( yMIN \) is a negative{!} number (-1 \( \leq \) yMIN < 0) which specifies the lower bound on the yp values selected to define the \( MLR \) exponent spline.

For each long-range term read power \( MMLR(i) \) & coefficient \( CmVAL(i) \).

For special Aubert-Frecon case, \( NCMM \) = 6, \( MMLR \) = {3,0,6,6,8,8} and coefficients are: \( CmVAL(1) = C3(sig) \), \( CmVAL(2) = ASO \), \( CmVAL(3) = C6(Sig) \), \( CmVAL(4)= C6(\pi) \), \( CmVAL(5)= C8(sig) \) & \( CmVAL(6)= C8(\pi) \).

Subroutine to calculate potential function value \( YC \) at distance:

\[ RDIST= RTP(IDAT) \]

and its partial derivatives with respect to the various potential parameters.

If \( IDAT \leq \) 1, then the subroutine will generate a new set of internal potential variables.

If \( IDAT \) > 1, then the subroutine will use SAVED values.
